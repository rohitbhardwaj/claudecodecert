# Claude-Assisted Refactor Review Checklist

```text
[ ] Did behavior change?
[ ] Are public error codes preserved?
[ ] Are failure paths tested?
[ ] Were any tests removed?
[ ] Were any assertions weakened?
[ ] Are logs still useful and safe?
[ ] Is the PR small and focused?
[ ] Was Claude usage disclosed?
[ ] Is rollback documented?
[ ] Are owner approvals identified?
```
