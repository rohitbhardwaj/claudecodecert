# Prompt Quality Rubric

## 5 — Excellent

Prompt includes:

```text
do not edit yet
analysis-first task
specific files to inspect
behavior to preserve
failure paths and tests
risk areas
approval owners
rollback considerations
wait for approval
```

## 4 — Strong

Prompt includes most required sections but misses one risk or approval area.

## 3 — Acceptable

Prompt is analysis-first but does not strongly protect tests or public behavior.

## 2 — Weak

Prompt asks for implementation too early or lacks behavior preservation.

## 1 — Poor

Prompt resembles the original vague prompt and permits test weakening.
