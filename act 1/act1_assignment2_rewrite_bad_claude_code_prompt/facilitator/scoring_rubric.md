# Scoring Rubric

## 5 — Excellent

Prompt is analysis-first, preserves behavior, protects tests, identifies risks and approvals, includes rollback, and waits for approval.

## 4 — Strong

Prompt includes most governance elements but misses one detail.

## 3 — Acceptable

Prompt is safer than original but does not fully protect behavior or tests.

## 2 — Weak

Prompt still asks for implementation too early.

## 1 — Poor

Prompt repeats the vibe coding pattern.
