# Prompt Review Checklist

Before using a Claude Code prompt on production code, check:

```text
[ ] Does the prompt say do not edit yet?
[ ] Does it ask for analysis first?
[ ] Does it name files/context to inspect?
[ ] Does it define behavior to preserve?
[ ] Does it protect tests?
[ ] Does it prohibit weakening/removing tests?
[ ] Does it identify risk areas?
[ ] Does it identify approval owners?
[ ] Does it require rollback considerations?
[ ] Does it wait for approval?
```
