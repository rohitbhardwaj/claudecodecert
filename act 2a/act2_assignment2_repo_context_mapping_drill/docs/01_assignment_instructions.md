# Assignment Instructions

## Assignment

Repo Context Mapping Drill

## Scenario

A team wants Claude Code to help with this task:

```text
Add support for enterprise-only coupons during checkout.
```

This may affect:

```text
web checkout
mobile checkout
batch invoice checkout
coupon policy
enterprise customer policy
pricing service
API response
tests
logging
rollback
```

## Architect Task

Before Claude proposes an implementation plan, decide which repo areas must be included in context.

## Key Question

```text
What must Claude inspect before it is allowed to propose changes?
```

## Key Lesson

```text
Before Claude Code edits the repo, architects must decide what context the repo must provide.
```
