# Act 2 Assignment 2 — Repo Context Mapping Drill

## Theme

Before Claude Code edits the repo, architects must decide what context the repo must provide.

## Scenario

A team wants Claude Code to help with this task:

```text
Add support for enterprise-only coupons during checkout.
```

This may affect:

```text
web checkout
mobile checkout
batch invoice checkout
coupon policy
enterprise customer policy
pricing service
API response
tests
logging
rollback
```

## Architect Task

Participants decide which files Claude must inspect before proposing an implementation plan.

## Key Lesson

```text
Before Claude Code edits the repo, architects must decide what context the repo must provide.
```
