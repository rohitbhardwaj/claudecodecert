# Assignment Instructions

## Assignment

Design the Approval Checkpoint

## Scenario

The team wants Claude Code to help with this task:

```text
Implement partial refunds for cancelled orders.
```

This may affect:

```text
CancellationService
RefundPolicy
PaymentGatewayClient
OrderRepository
Checkout API response
RefundServiceTest
PaymentGatewayClientTest
Logging policy
Rollback runbook
CODEOWNERS
```

Claude should not implement until the team approves the plan.

## Architect Task

Design the approval checkpoint Claude must pass before editing.

## Output

For each approval area, define:

```text
what Claude must provide
who approves
why the approval matters
```

## Key Lesson

```text
Approval is not a meeting. It is evidence that the right risks were understood before code changed.
```
