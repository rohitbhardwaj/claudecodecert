# Act 4 Assignment 2 — Design the Approval Checkpoint

## Theme

Approval is not a meeting. It is evidence that the right risks were understood before code changed.

## Scenario

The team wants Claude Code to help with this task:

```text
Implement partial refunds for cancelled orders.
```

This may affect:

```text
CancellationService
RefundPolicy
PaymentGatewayClient
OrderRepository
Checkout API response
RefundServiceTest
PaymentGatewayClientTest
Logging policy
Rollback runbook
CODEOWNERS
```

Claude should not implement until the team approves the plan.

## Architect Task

Participants design the approval checkpoint Claude must pass before editing.

## Key Lesson

```text
Approval is not a meeting. It is evidence that the right risks were understood before code changed.
```
